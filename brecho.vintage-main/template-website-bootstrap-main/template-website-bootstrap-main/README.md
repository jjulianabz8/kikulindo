# Template Portfólio Bootstrap 5

## 📚 Sobre o Projeto

Este projeto foi desenvolvido como material didático para alunos do Ensino Médio que estão iniciando seus estudos em desenvolvimento web utilizando HTML, CSS e Bootstrap 5.

O objetivo é fornecer uma página pronta contendo diversos componentes do Bootstrap para que os alunos possam analisar o código, modificar os elementos e aprender na prática como construir interfaces modernas e responsivas.

---

## 🚀 Componentes Utilizados

Neste template você encontrará exemplos dos seguintes componentes do Bootstrap 5:

* Navbar (Menu de Navegação)
* Carousel (Carrossel de Imagens)
* Sistema de Grid (Rows e Columns)
* Cards
* Formulários
* Accordion (Perguntas Frequentes)
* Botões
* Containers
* Classes Utilitárias
* Responsividade

Além disso, há exemplos de CSS personalizado para complementar a estilização do Bootstrap.

## 🎯 Objetivos de Aprendizagem

Ao estudar este template, você poderá:

* Entender a estrutura básica de uma página web.
* Aprender a utilizar o Bootstrap através de CDN.
* Criar layouts responsivos.
* Trabalhar com componentes prontos do framework.
* Personalizar estilos utilizando CSS.
* Desenvolver um portfólio próprio.

---

## 🛠️ Como Utilizar

1. Faça o download do projeto.
2. Abra a pasta no Visual Studio Code.
3. Execute o arquivo `index.html` no navegador.
4. Analise os componentes existentes.
5. Modifique textos, imagens e cores.
6. Crie novas seções e componentes.
7. Transforme o template em seu próprio portfólio.

---

## 💡 Desafios Propostos

### Nível 1

* Altere o título da página.
* Troque as imagens do carrossel.
* Modifique as cores do menu.

### Nível 2

* Adicione mais um card de projeto.
* Inclua novos itens no menu.
* Personalize o formulário de contato.

### Nível 3

* Crie uma seção "Habilidades".
* Adicione ícones utilizando Bootstrap Icons.
* Desenvolva uma versão totalmente personalizada do portfólio.
